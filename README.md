# emoji-snippets.vim

emoji snippets for vim, working with [YouCompleteMe](https://github.com/ycm-core/YouCompleteMe) for completion.

## Supported emoji

| snippets trigger | emoji |
| :--------------- | :---- |
| emjsmile         | 😄    |
| emjcheck         | ✅    |
| emjx             | ❌    |
| emjcircle        | ⚪️   |
| emjsquare        | 🔲    |
